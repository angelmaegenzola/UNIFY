## About

Unify is a web-based student organization management system designed to simplify how campus clubs and organizations operate. From planning events to managing membership roles, Unify brings everything your org needs into one place.

Live site: [https://unify.bsit2c.site](https://unify.bsit2c.site)

---

## Features

- **Event Planning** — Create, schedule, and manage organization events
- **Role Assignment** — Assign and manage member roles within your organization
- **Member Recruitment** — Streamline the process of onboarding new members
- **Organization Dashboard** — Get a real-time overview of your org's activities
- **Announcements** — Keep all members updated and in sync
- **Attendance Monitoring** — Track event attendance via QR code scanning or manual check-in; records are accessible to officers only
- **Sign Up / Sign In** — Secure user authentication for members and admins

## Getting Started

### Prerequisites

- PHP 7.4 or higher
- MySQL
- Composer
- A local server environment (XAMPP)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/angelmaegenzola/unify.git
   cd unify
   ```

2. Move the project folder into the XAMPP `htdocs` directory.

3. Start Apache and MySQL from XAMPP Control Panel.

4. Create a new database in phpMyAdmin.

5. Import the provided SQL database file (unify_db).

6. Configure the database connection settings.

7. Open your browser and run:

```text
http://localhost/unify
```

## Usage

1. Visit the landing page and click **Get Started** to register
2. Sign in to access your organization dashboard
3. Create or join a student organization
4. Start managing events, members, and roles
5. Track attendance by generating a QR code for an event or manually marking members present — only officers can view attendance records

## Attendance Monitoring
Unify includes a built-in attendance tracking system for organization events.

**How it works**

QR Code Check-in

-Each student is assigned a unique personal QR code generated by the system
-During an event, the student presents their QR code to the officer
-The officer scans the student's QR code using the Unify attendance scanner
-The system automatically records the student as present with a timestamp

**Manual Check-in**

Officers can also manually mark individual members as present, absent, or excused directly from the attendance panel — useful when a student forgets or cannot present their QR code

**Access Control**

Attendance records are restricted to officers only. Regular members cannot view attendance data.

**What gets tracked**

-Member name and ID
-Event name and date
-Check-in method (QR or manual)
-Attendance status (present / absent / excused)
-Timestamp of check-in
